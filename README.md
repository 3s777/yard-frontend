# Project for learning frontend "Compass"

In project we use

  - ReactJS
  - React Router
  - Webpack
  - Styled components
  - Yarn
  - Eslint
  - Eslint-config-airbnb
  - Now.sh
  - Travis-ci.org
  - Flow.org
  - Es6
  - Html5
  - Css3

# How it work

  - yarn start (run application)
  - yarn prepush (check eslint rules)
  - yarn build (local build)
  - now build (deploy on now.sh)
