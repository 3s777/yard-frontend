# Project for learning frontend "Compass"

In project we use

  - ReactJS
  - React Router
  - Webpack
  - Styled components
  - Yarn
  - Eslint
  - Eslint-config-airbnb
  - Now.sh
  - Travis-ci.org
  - Flow.org
  - Es6
  - Html5
  - Css3

## How it work

### Run application
```sh
$ yarn start
```
### Check eslint rules
```sh
$ yarn lint
```
### Local build
```sh
$ yarn build
```
### Deploy on now.sh
```sh
$ now build
```
